Directions

Use the q-slider to explore different feed conditions in a distillation column:

  • q < 0: Superheated vapor feed
  • q = 0: Saturated vapor feed
  • 0 < q < 1: Partially vaporized feed
  • q = 1: Saturated liquid feed
  • q > 1: Subcooled liquid feed

Watch how the internal vapor and liquid flows change as you adjust the feed condition.

About

This simulation demonstrates the effect of feed thermal condition (q-value) on internal flows in a distillation column feed stage.

Based on Section 7.2.3 of "Separation Process Principles" by Seader, Henley, and Roper.

Phase 1: Basic structure with static column visualization and q-slider functionality.

Technical Details
Key Equations:

q-parameter: q = (L̅ - L) / F

Material Balance: F + V + L = V̅ + L̅

q-Line: y = (q/(q-1)) × x - (zF/(q-1))

Feed Conditions:
q Value Feed Condition Effect
q < 0 Superheated Vapor Vaporizes reflux
q = 0 Saturated Vapor Adds vapor only
0 < q < 1 Partially Vaporized Adds vapor and liquid
q = 1 Saturated Liquid Adds liquid only
q > 1 Subcooled Liquid Condenses vapor
